# Recovery Radar SEO Growth Suite

The ultimate master build adds a complete SEO and keyword workflow for Infinite Pieces AI.

## Routes

- `/keyword-radar` — original keyword radar upgraded with the full suite.
- `/seo-command-center` — direct full-screen SEO command center.

## What it does

1. **Keyword Planner CSV Import**
   - Paste or upload Google Ads Keyword Planner CSV data.
   - Reads keyword, average monthly searches, competition, competition index and top-of-page bid ranges.
   - Scores each keyword by intent, volume, bid signal and competition.

2. **SEO Page Builder**
   - Select a keyword and generate URL slug, SEO title, H1, meta description, outline, FAQ, CTAs, internal links and ad headlines.

3. **SERP Checker**
   - Uses Google Custom Search API when configured.
   - Returns top results, snippets, rough total result count and content gap recommendations.

4. **Keyword-to-Page Map**
   - Groups keywords into canonical URLs so the site does not create duplicate pages that compete with each other.

5. **Content Calendar**
   - Generates a 30-day plan across landing pages, blogs, LinkedIn posts, email nurture and ad groups.

6. **Local SEO Mode**
   - Generates city/state keyword variations and landing page ideas.

7. **Competitor Radar**
   - Builds competitor/alternative keyword targets for CentralReach, Rethink, Motivity, Catalyst, ATrack and Hi Rasmus.

8. **Lead + Keyword Bridge**
   - Maps lead-search queries to SEO pages and outreach angles.

9. **Website Audit**
   - Scans any public URL for title, meta, H1, recovery CTAs, Provider Portal CTA and no-migration positioning.

10. **Search Console CSV Import**
   - Paste Google Search Console performance export and prioritize high-impression/low-CTR opportunities.

## Recommended workflow

1. Run Google Ads Keyword Planner with ABA seed keywords.
2. Export CSV.
3. Import into `/seo-command-center`.
4. Review Priority List.
5. Build the top landing pages first:
   - `/centralreach-alternative`
   - `/aba-emr-alternative`
   - `/aba-cancellation-management`
   - `/rbt-callout-coverage`
   - `/how-to-open-an-aba-clinic`
6. Use Website Audit after publishing.
7. Import Search Console CSV after the site starts getting impressions.
