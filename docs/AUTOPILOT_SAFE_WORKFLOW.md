# Founder Autopilot Safe Workflow

This workflow is designed for research, organization, drafting, and human review.

The system should help prepare work, not bypass human judgment or platform rules.

Core flow:

1. Find public source.
2. Review source manually.
3. Save public business-level context.
4. Score the opportunity with visible reasons.
5. Prepare a human-reviewed note.
6. Require approval before use.
7. Log outcomes and suppression signals.

Guardrails:

- Public sources only.
- No PHI.
- No hidden identity.
- No platform bypassing.
- No automatic external action.
- Human review required.
